# Alonso & Espinosa · Consultoría Operativa

Esta habilidad permite gestionar el ecosistema digital de Alonso & Espinosa, asegurando que la visibilidad y la operativa interna sigan el **Método A&E**.

## 🎯 Cuándo usar esta habilidad
- Cuando el usuario pida centralizar nuevos activos (Excel, PDFs) en Notion.
- Cuando se necesite actualizar la web oficial en GitHub Pages.
- Cuando se requiera crear contenido para redes sociales (LinkedIn/Facebook) que atraiga clientes.
- Cuando se inicie un nuevo proyecto de consultoría para un cliente.

## 📋 Flujo de Trabajo Principal

### 1. Centralización en Notion
- **CRM de Clientes:** Usa la plantilla en `templates/notion_crm_template.md` para crear o actualizar el tablero de gestión de clientes.
- **Activos:** Vincula siempre los Excel sectoriales (Cerebro Solar, Alérgenos, Presupuesto Obra) en el espacio de trabajo del cliente.
- **Notas Estratégicas:** Consulta `references/metodo_ae.md` para asegurar que el discurso comercial sea coherente (Orden > Tecnología).

### 2. Visibilidad en la Web (GitHub)
- **Repositorio:** `alauriela/alonsoespinosa-web`.
- **Estilo:** Mantener siempre los colores corporativos (Azul Oscuro: `#2E4A5A`, Celeste: `#5AD1E5`) y tipografías (Lora, DM Sans).
- **Sección de Herramientas:** Actualizar el `index.html` cuando se creen nuevos activos operativos para mostrarlos como prueba de autoridad.

### 3. Atracción en Redes Sociales
- **LinkedIn (B2B):** Foco en rentabilidad, márgenes y reducción de carga administrativa (alérgenos, presupuestos).
- **Facebook (Local):** Foco en hostelería y reformas locales.
- **Estructura de Post:** Problema (Caos) -> Prueba (Vuestros Excel) -> Solución (Método A&E) -> CTA (Llamada de 20 min).

### 4. Creación de Materiales Comerciales
- **Presentaciones:** Usa la guía en `references/guias_presentaciones.md` para diseñar diapositivas que muestren prototipos de digitalización (Notion, KDS, Excel).
- **Prototipos:** Recrea interfaces de Notion en las diapositivas para demostrar la operativa real antes de la implantación.

## 📂 Recursos Disponibles
- `references/metodo_ae.md`: Resumen del método, fases y filosofía.
- `references/guias_presentaciones.md`: Estándares visuales y narrativos para crear diapositivas comerciales.
- `references/digitalizacion_tecnica.md`: Guía paso a paso para conectar Excel, Make y Notion.
- `references/guia_negocio_ae.md`: Modelo de negocio, costes y estrategia comercial.
- `templates/notion_crm_template.md`: Estructura base para el CRM en Notion.

## ⚠️ Reglas de Oro
- **No vender IA como fin:** Siempre como medio para el orden operativo.
- **Protección Interna:** No revelar los pasos técnicos de implantación en documentos públicos.
- **Simplicidad:** La web debe cargar rápido y ser visualmente ligera.
